Step-by-Step Instructions for Upload Data File

  1. Upload Data:
    • Select a file from your computer containing spectrophotometer scan results.
    • Ensure the file is saved in a compatible format such as CSV or Excel.
    • Open the file in Excel to review the data.
    • Delete any columns not containing data within the wavelength range of 380nm to 750nm.
    • Ensure that the file only contains data for every 10nm. Do not include intermediate values.
    • Save the file as a txt file with the necessary data range containing the data for every 10nm.
    • Click "Submit" to upload the prepared file for processing.

Ensure all wavelengths from 380nm to 750nm are covered with their respective transmittance and absorption values. This data is crucial for calculating lens ratings.
